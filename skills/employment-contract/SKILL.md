# Employment Contract Generator — FM-HR-001

สร้างสัญญาจ้างพนักงานครบถ้วนจากข้อมูลที่ได้รับ

## ขั้นตอนการทำงาน

### Step 1 — รวบรวมข้อมูล

ดึงข้อมูลจากสิ่งที่ผู้ใช้บอกมาก่อน จากนั้นถามข้อมูลที่ยังขาดเป็นชุดเดียว (ถามครั้งเดียว ไม่ถามซ้ำ)

**ข้อมูลที่ต้องการ** แบ่งเป็น 2 กลุ่ม:

#### กลุ่ม A — ข้อมูลบริษัท (ถามเฉพาะถ้าไม่รู้)
| Field | ตัวอย่าง |
|-------|---------|
| ชื่อบริษัทไทย | บริษัท ฮิวแมนเอไอ จำกัด |
| ชื่อบริษัทอังกฤษ | HumanAI Co., Ltd. |
| ที่อยู่บริษัท | 555/42 ม.2 ต.หนองจ๊อม อ.สันทราย จ.เชียงใหม่ 50210 |
| เลขภาษี | 0505568019768 |
| เบอร์โทรบริษัท | 061-797-8899 |
| ชื่อ-ตำแหน่งผู้ลงนาม (ผู้ว่าจ้าง) | นายปริวรรตน์ อรุโณทยานันท์ / กรรมการผู้จัดการ |
| วันที่ทำสัญญา | 1 เมษายน 2569 |

#### กลุ่ม B — ข้อมูลพนักงาน (จำเป็นทุกข้อ)
| Field | หมายเหตุ |
|-------|---------|
| ชื่อ-นามสกุล (ไทย) | |
| ชื่อ-นามสกุล (อังกฤษ) | ถ้าไม่มีใส่ ............... |

#### กลุ่ม C — ข้อมูลตำแหน่ง
| Field | หมายเหตุ |
|-------|---------|
| ตำแหน่งงาน | |
| ฝ่าย / แผนก | |

#### กลุ่ม E — ค่าตอบแทน (ถามให้ครบ ข้อมูลนี้สำคัญมาก)

- รายการ 1–6 (ขอให้ระบุอย่างน้อย 1 ข้อ ที่เหลือถ้าไม่มีให้ว่างไว้)

### Step 2 — ตรวจสอบข้อมูลที่ขาด

ก่อน build ให้ตรวจว่าข้อมูลใดที่ไม่ได้รับ:
- **ข้อมูลสำคัญมาก** (ชื่อ, ตำแหน่ง, เงินเดือน, วันเริ่มงาน, ประเภทสัญญา) → **ต้องถาม** ก่อน build
- **ข้อมูลรอง** (E-mail, สัญชาติ, เลขบัญชี ฯลฯ) → ใส่ `...............` แทน และแจ้งผู้ใช้ว่าต้องกรอกเพิ่ม

---

### Step 3 — Build ไฟล์ .docx

อ่านไฟล์ reference ก่อน build:
```bash
cat /mnt/skills/user/employment-contract/references/build-template.js
```

จากนั้น copy ไปที่ `/home/claude/` แก้ข้อมูล และ build:
```bash
cp /mnt/skills/user/employment-contract/references/build-template.js /home/claude/build_contract_fill.js
# แก้ข้อมูลใน DATA BLOCK
node /home/claude/build_contract_fill.js
python3 /mnt/skills/public/docx/scripts/office/validate.py /home/claude/FM-HR-001-filled.docx
cp /home/claude/FM-HR-001-filled.docx /mnt/user-data/outputs/สัญญาจ้าง-[ชื่อพนักงาน].docx
```

### Step 4 — แจ้งผลและ checklist

หลัง build เสร็จ แจ้งผู้ใช้:
1. ส่วนที่กรอกครบแล้ว
2. ช่อง `...............` ที่ยังต้องกรอกเพิ่มเองใน Word
3. เอกสารที่ต้องเตรียมแนบ (แนบ 1–5)

## Font & Style

- **Font**: Angsana New ทุกส่วน
- **ขนาด body**: 12pt (24 half-points)
- **ขนาด header/footer**: 10pt (20 half-points)
- **ขนาด title ใหญ่**: 28pt (56 half-points)
- **palette**: monochrome (nearBlack / darkGray / midGray / lineGray)
- **ไม่มีกรอบสี่เหลี่ยมตกแต่ง** — ใช้ตารางเฉพาะส่วนกรอกข้อมูล
- **ไม่มีโลโก้** — header แสดงชื่อบริษัท + ที่อยู่ + เลขภาษีเท่านั้น
