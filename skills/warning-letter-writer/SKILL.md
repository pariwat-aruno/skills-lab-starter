# Warning Letter Writer Skill

เขียนหนังสือเตือนพนักงานภาษาไทยแบบมืออาชีพ มีผลทางกฎหมาย อ้างอิง พ.ร.บ.คุ้มครองแรงงาน และนำไปใช้ได้จริงในศาลแรงงาน

---

## รูปแบบการเขียน — "เรียงความทางกฎหมาย"

หนังสือเตือนใช้รูปแบบ **เรียงความเป็นย่อหน้าต่อเนื่อง** ไม่แบ่งเป็นหัวข้อย่อย
ไหลตามลำดับ "กฎ → เกิดอะไร → สอบสวนยังไง → สรุปยังไง → เตือนอะไร → ผลของการฝ่าฝืน"
เหมือนคำพิพากษาศาลแรงงาน: ข้อเท็จจริง → ข้อกฎหมาย → คำวินิจฉัย

---

## หลักการเขียน 10 ข้อ

### หลัก 1 — โครงสร้าง "เล่าเรื่องตามลำดับเวลา"

เขียนเป็นเรียงความที่ไหลต่อเนื่อง โดยเรียงย่อหน้าตามลำดับนี้:

| ลำดับ | เนื้อหาย่อหน้า | หน้าที่ |
|-------|----------------|---------|
| 1 | อ้างอิงข้อบังคับที่เกี่ยวข้อง | ปูฐานว่ามีกฎอยู่แล้ว |
| 2 | ที่มาของเรื่อง / เหตุการณ์ | อธิบายว่าเรื่องเริ่มจากตรงไหน |
| 3 | ข้อเท็จจริง + หลักฐาน | เรียงรายละเอียดตาม fact |
| 4 | การดำเนินการก่อนหน้า (ตักเตือนวาจา / สอบสวน) | แสดง Progressive Discipline |
| 5* | ผลการสอบสวน + การชั่งน้ำหนักพยานหลักฐาน | เฉพาะกรณีที่มีการสอบสวน |
| 6 | คำเตือน + สิ่งที่ต้องปฏิบัติ | ระบุชัดว่าต้อง "ทำอะไร" |
| 7 | ผลของการไม่ปฏิบัติตาม | อ้าง ม.119(4) + อายุ 1 ปี |
| 8 | สิทธิชี้แจง | Right to be heard |
| 9 | จำนวนสำเนา | 3 ฉบับ |

*ย่อหน้า 5 ใช้เฉพาะกรณีที่ต้องมีการสอบสวน (นินทา ทุจริต ทะเลาะ ฯลฯ)

### หลัก 2 — "ปูกฎก่อน ชี้ผิดทีหลัง"

- เปิดย่อหน้าแรกด้วยการยกข้อบังคับที่เกี่ยวข้อง **ทุกข้อ**
- ระบุ "หมวดที่ X ข้อ X.X" + ยกถ้อยคำข้อบังคับเป็น **ตัวเอียง (italic)**
- อ้างข้อบังคับอย่างน้อย 2-3 ข้อ ยิ่งครบยิ่งมีน้ำหนัก
- อ้าง พ.ร.บ.คุ้มครองแรงงาน มาตรา 108 ว่าบริษัทจัดทำข้อบังคับตามกฎหมาย
- ระบุว่าพนักงานได้รับทราบข้อบังคับแล้ว (ลงนามรับเมื่อวันเข้าทำงาน)

**ตัวอย่างการอ้าง:**
> ข้อบังคับเกี่ยวกับการทำงาน **หมวดที่ 3 ว่าด้วยเวลาทำงานปกติ ข้อ 3.1** กำหนดไว้ว่า *"เวลาทำงานปกติของพนักงานคือวันจันทร์ถึงวันเสาร์ ตั้งแต่เวลา 08:30 น. ถึง 17:30 น. ..."*

### หลัก 3 — ข้อเท็จจริงต้อง "ตรวจสอบได้" ทุกประโยค

ทุกครั้งที่เขียนว่าพนักงานทำอะไร ต้องระบุ **5 องค์ประกอบ**:

| องค์ประกอบ | ตัวอย่าง |
|-----------|---------|
| **ทำอะไร** | สแกนนิ้วเข้างาน / กล่าวถ้อยคำ |
| **เมื่อไร** | วันจันทร์ที่ 3 มีนาคม 2569 เวลา 09:47 น. |
| **ที่ไหน** | ณ สถานที่ทำงาน / ห้องพักร่วม |
| **หลักฐาน** | ระบบสแกนลายนิ้วมือ / พยานบุคคล 4 คน / ข้อความ LINE |
| **ผิดข้อไหน** | ข้อบังคับหมวดที่ 3 ข้อ 3.2 |

**ห้ามเขียนกว้าง** เช่น "มาสายบ่อย" "นินทาเพื่อนร่วมงาน" → ต้องระบุรายละเอียดเสมอ

### หลัก 4 — แสดง "Progressive Discipline"

- เขียนให้เห็นว่าก่อนออกหนังสือเตือน บริษัทให้โอกาสแก้ไขแล้ว
- ระบุ: **ชื่อคนที่เตือน + วันที่เตือน + ครั้งที่เท่าไร + พนักงานรับทราบหรือไม่**
- ลำดับขั้น: ตักเตือนวาจา ครั้งที่ 1 → ครั้งที่ 2 → หนังสือเตือน
- หรือ: รับเรื่องร้องเรียน → สอบสวน → หนังสือเตือน
- ปิดด้วยประโยคว่า "พนักงานยังคงไม่ปรับปรุงแก้ไขพฤติกรรม"

### หลัก 5 — กรณีมีการสอบสวน ต้องเขียนครบ 5 ขั้นตอน

ใช้เมื่อเป็นกรณี: นินทา, ทุจริต, ทะเลาะ, คุกคามทางเพศ, ขโมยทรัพย์สิน, ปลอมเอกสาร ฯลฯ

| ขั้นตอน | สิ่งที่ต้องเขียน |
|--------|-----------------|
| 5.1 แต่งตั้งคณะกรรมการ | ชื่อ ตำแหน่ง เหตุผล + อ้างข้อบังคับหมวดสอบสวน + กรรมการ 3 คนขึ้นไป + ไม่มีส่วนได้เสีย |
| 5.2 รวบรวมพยานหลักฐาน | แยก 3 ประเภท: พยานบุคคล (ชื่อ ตำแหน่ง ให้ถ้อยคำแยกจากกัน สอดคล้องต้องกัน) / พยานเอกสาร (ข้อความ LINE ภาพถ่าย บันทึก) / พยานวัตถุ (CCTV ระบบสแกน) |
| 5.3 ให้โอกาสชี้แจง | แจ้งข้อกล่าวหาล่วงหน้ากี่วัน + อ้างข้อบังคับ + พนักงานชี้แจงว่าอะไร + มีผู้แทนร่วมหรือไม่ |
| 5.4 ชั่งน้ำหนักพยานหลักฐาน | อธิบายว่าข้อแก้ตัว **ทำไมฟังไม่ขึ้น** ด้วยเหตุผลเชิงตรรกะ ไม่ใช่แค่ "ไม่เชื่อ" |
| 5.5 มติคณะกรรมการ | เอกฉันท์หรือเสียงข้างมาก + สรุปว่า "ข้อกล่าวหาฟังขึ้น" |

**สำคัญ:** ย่อหน้า "ชั่งน้ำหนัก" ต้องเขียนแบบ **หักล้างทีละข้อ** เช่น:
> คำชี้แจงที่ว่า "พูดเล่นๆ" ไม่สามารถรับฟังได้ เนื่องจากถ้อยคำมีเนื้อหาเฉพาะเจาะจง...
> ส่วนคำชี้แจงที่ว่า "ไม่ได้หมายถึงใคร" ก็ไม่อาจรับฟังได้เช่นกัน เพราะบริบท...

### หลัก 6 — คำเตือนต้องบอก "สิ่งที่ต้องทำ" ไม่ใช่แค่ "ห้ามทำอีก"

- เขียนเป็น (1) (2) (3) ในเนื้อย่อหน้า (ไม่แยกเป็น bullet)
- ระบุ **สิ่งที่ต้องปฏิบัติ** ให้เฉพาะเจาะจง
- ❌ "ห้ามมาสาย" → ✅ "มาถึงสถานที่ทำงานไม่เกิน 08:30 น. ทุกวันทำงาน"
- ❌ "ห้ามนินทา" → ✅ "หยุดกล่าวถ้อยคำที่สร้างความเสื่อมเสียต่อเพื่อนร่วมงานทุกคนทันที"
- ข้อสุดท้ายปิดด้วย "ปฏิบัติตามข้อบังคับเกี่ยวกับการทำงานอย่างเคร่งครัด"

### หลัก 7 — ผลของการไม่ปฏิบัติตาม อ้าง ม.119(4) ชัดเจน

ย่อหน้านี้ต้องมี **3 องค์ประกอบ**:

1. **เตือนว่าจะเลิกจ้างไม่จ่ายค่าชดเชย** — "บริษัทฯ มีความจำเป็นที่จะต้องดำเนินมาตรการทางวินัยขั้นสูงสุด ซึ่งรวมถึงการเลิกจ้างโดยไม่จ่ายค่าชดเชย"
2. **ยกถ้อยคำ ม.119(4) เป็นตัวเอียง** — "ฝ่าฝืนข้อบังคับเกี่ยวกับการทำงาน ระเบียบ หรือคำสั่งของนายจ้างอันชอบด้วยกฎหมายและเป็นธรรม และนายจ้างได้ตักเตือนเป็นหนังสือแล้ว..."
3. **อายุ 1 ปี ตาม ม.119 วรรคท้าย** — "หนังสือเตือนฉบับนี้มีผลบังคับเป็นระยะเวลา 1 ปี นับแต่วันที่พนักงานได้รับทราบ"

### หลัก 8 — เปิดช่อง Right to be heard

- "สามารถจัดทำหนังสือชี้แจงเป็นลายลักษณ์อักษรส่งถึงฝ่ายบริหารงานบุคคลได้ภายใน 7 วัน"
- "บริษัทฯ จะพิจารณาข้อชี้แจงของพนักงานด้วยความเป็นธรรม"
- แม้กฎหมายไม่บังคับ แต่ศาลแรงงานพิจารณาเสมอ

### หลัก 9 — ลายมือชื่อ 3 ฝ่าย + แผนสำรอง

ช่องลงนาม 3 ส่วน:

| ลำดับ | ผู้ลงนาม | หมายเหตุ |
|-------|---------|---------|
| 1 | ผู้ออกหนังสือเตือน | ต้องเป็นนายจ้าง/ผู้มีอำนาจ ไม่ใช่แค่หัวหน้างาน |
| 2 | พนักงานผู้รับ | เพิ่มข้อความ "ข้าพเจ้าได้รับหนังสือเตือนฉบับนี้แล้ว และได้อ่านทำความเข้าใจข้อความทั้งหมดแล้ว" เป็นตัวเอียง |
| 3 | พยาน 1-2 คน | เป็นบุคคลที่สามารถเป็นพยานได้ |

**หมายเหตุ HR ท้ายเอกสาร:**
- กรณีพนักงานปฏิเสธเซ็น → บันทึก "พนักงานปฏิเสธลงนามรับ" + พยาน 2 คนรับรอง
- หนังสือเตือนยังมีผลแม้พนักงานไม่ลงนาม
- ระบุเอกสารประกอบที่ต้องเก็บรวมไว้ในแฟ้ม (ถ้ามีการสอบสวน)

### หลัก 10 — โทนภาษา "เป็นทางการแต่ไม่ข่มขู่"

| ✅ ใช้ | ❌ ห้ามใช้ |
|--------|-----------|
| "พนักงานได้กระทำการฝ่าฝืน" | "พนักงานจงใจก่อปัญหา" |
| "ข้อเท็จจริงปรากฏว่า" | "พนักงานโกหก" |
| "คำชี้แจงไม่สามารถรับฟังได้" | "ข้ออ้างเป็นเรื่องไร้สาระ" |
| "หวังว่าจะเป็นโอกาสให้ปรับปรุงตนเอง" | "นี่คือโอกาสสุดท้าย" |
| "มีความจำเป็นต้องดำเนินมาตรการ" | "จะไล่ออก" |

- ภาษาราชการ สุภาพ เป็นกลาง
- ย่อหน้าท้ายมีประโยค "หวังดี" เสมอ
- ถ้ามีความผิดทางอาญาเกี่ยวข้อง เตือนให้ทราบแต่ระบุว่า "ไม่ประสงค์ให้บานปลาย"

## ประเภทความผิดที่พบบ่อย + ข้อบังคับที่ต้องอ้าง

| ประเภทความผิด | ข้อบังคับที่ต้องอ้าง | ต้องสอบสวนหรือไม่ | กฎหมายอาญาเกี่ยวข้อง |
|--------------|---------------------|-------------------|---------------------|
| มาสาย/ขาดงาน | หมวดเวลาทำงาน + หมวดวินัย | ไม่จำเป็น (มีหลักฐานระบบชัด) | ไม่มี |
| นินทา/ใส่ร้าย | หมวดวินัย (ห้ามใส่ร้าย + ห้ามสร้างแตกแยก + รักษาสามัคคี) | **ต้องสอบสวน** | ม.326, 328 หมิ่นประมาท |
| ทุจริต/ยักยอก | หมวดวินัย (ซื่อสัตย์สุจริต) | **ต้องสอบสวน** | ม.352 ยักยอก / ม.354 |
| ทะเลาะ/ทำร้ายร่างกาย | หมวดวินัย (ห้ามใช้ความรุนแรง) | **ต้องสอบสวน** | ม.295 ทำร้ายร่างกาย |
| ละทิ้งหน้าที่ | หมวดหน้าที่ + หมวดวินัย | ไม่จำเป็น | ไม่มี |
| ทำผิดกฎความปลอดภัย | หมวดความปลอดภัย + หมวดวินัย | แล้วแต่ความร้ายแรง | พ.ร.บ.ความปลอดภัยฯ |
| เปิดเผยความลับ | หมวดวินัย (รักษาความลับ) + สัญญาจ้าง NDA | **ต้องสอบสวน** | พ.ร.บ.ความลับการค้า |
| คุกคามทางเพศ | หมวดวินัย + นโยบายป้องกันคุกคาม | **ต้องสอบสวน** | ม.397 / พ.ร.บ.คุ้มครองแรงงาน ม.16 |

## ขั้นตอนการทำงาน

### Step 1 — เก็บข้อมูลจากผู้ใช้

ข้อมูลที่ต้องมี (ถามถ้าไม่มีในบริบท):

| ข้อมูล | จำเป็น | หมายเหตุ |
|--------|--------|---------|
| ประเภทความผิด | ✅ | มาสาย, นินทา, ทุจริต ฯลฯ |
| ชื่อ-ตำแหน่ง-แผนก พนักงาน | ✅ | ถ้าไม่มีใช้ตัวอย่าง |
| ข้อเท็จจริง (วัน เวลา เหตุการณ์) | ✅ | ถ้าไม่มีสร้างตัวอย่างที่สมจริง |
| ข้อบังคับที่เกี่ยวข้อง | สร้างเอง | สมมติหมวด/ข้อที่เหมาะสม |
| มีการสอบสวนหรือไม่ | ตามประเภท | ดูตารางด้านบน |
| พนักงานชี้แจงว่าอะไร | ถ้ามีสอบสวน | สมมติคำชี้แจงที่สมจริง |
| ชื่อบริษัท | ถ้ามี | ถ้าไม่มีใช้ตัวอย่าง |
| output แชท หรือ .docx | ถ้ามี | default = .docx |

**ถ้าผู้ใช้ให้ข้อมูลน้อย** → สร้างตัวอย่างที่สมจริงครบทุกส่วนเลย ไม่ต้องถามทุกข้อ

### Step 2 — เขียนหนังสือเตือน

## Word Document Style

### การจัดรูปแบบ

| Element | วิธีทำ |
|---------|--------|
| ชื่อ "หนังสือเตือน" | size 48 bold nearBlack กลาง + rule() ใต้ |
| เลขที่ + วันที่ | ชิดขวา |
| เรียน + เรื่อง | ชิดซ้าย / Justified |
| เนื้อย่อหน้า | Justified, firstLine indent 600, mixed runs (bold สำหรับข้อบังคับ, italic สำหรับถ้อยคำกฎหมาย) |
| ลายมือชื่อ | กลาง, เว้น sp(3) ระหว่างแต่ละฝ่าย |
| หมายเหตุ HR | rule() + ตัวเอียง midGray size 17 |

### Mixed Runs Pattern

ย่อหน้าหนึ่งมีหลาย TextRun ผสมกัน:
- ข้อความปกติ: `size: 20, color: darkGray`
- ชื่อข้อบังคับ/หมวด: `bold: true, color: darkGray`
- ถ้อยคำข้อบังคับ/กฎหมาย: `italics: true, color: darkGray`
- ข้อความเน้น: `bold: true, color: darkGray`

### Font & สี & Page Setup

→ ใช้ตาม humanai-docs skill ทุกประการ:
- Font: Niramit ทุกส่วน
- Palette: nearBlack 1A1A1A, darkGray 2D2D2D, midGray 595959, lineGray D4D4D4
- US Letter: 12240 × 15840, Margin: top/bottom 1080, left/right 1560
- Header/Footer: ใช้ makeHeader() / makeFooter() จาก hai_helpers.js

## ข้อควรระวัง

- **ห้ามเขียนข้อเท็จจริงกว้างเกินไป** — ต้องระบุวัน เวลา สถานที่ หลักฐาน ทุกครั้ง
- **ห้ามตัดสินพนักงานด้วยภาษารุนแรง** — ใช้ภาษาเป็นกลาง
- **ห้ามลืมอ้างข้อบังคับ** — หนังสือเตือนที่ไม่อ้างข้อบังคับศาลอาจตีความว่าไม่สมบูรณ์
- **ห้ามลืม ม.119(4) + อายุ 1 ปี** — องค์ประกอบสำคัญที่สุด
- **ห้ามลืมลายมือชื่อ 3 ฝ่าย** — โดยเฉพาะพยาน
- **กรณีต้องสอบสวน ห้ามข้ามขั้นตอน** — ต้องเขียนครบ 5 ขั้นตอน
- **ห้ามใช้ unicode bullet** ใน .docx → ใช้ numbering config
- **ห้ามใช้ `\n`** ใน TextRun → แยก Paragraph
- ตรวจสอบด้วย `validate.py` ก่อนส่งให้ผู้ใช้เสมอ
