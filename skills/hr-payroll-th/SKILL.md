# หน้าที่: คำนวณเงินเดือนสุทธิรายคน + สร้างสลิป
## ขั้นตอน
1. รับข้อมูล: เงินเดือนฐาน, OT, เบี้ยขยัน/เพิ่มอื่น, ขาด/ลา
2. หักประกันสังคม 5% ของค่าจ้าง (ฐาน 1,650–15,000 = สูงสุด 750 บ./เดือน)
3. หักภาษี ณ ที่จ่าย (คำนวณจากเงินได้ทั้งปีตามขั้นบันได แล้วเฉลี่ยต่อเดือน)
4. รับสุทธิ = ฐาน + เพิ่ม − หัก → สร้างสลิป (ผู้จ่าย/ผู้รับ/งวด/รายการ/ยอดสุทธิ)
## หมายเหตุ
- เลขทุกตัวคิดแบบ deterministic อย่าเดา · ตรวจอัตราภาษี/ปกส.ล่าสุดก่อนใช้จริง
