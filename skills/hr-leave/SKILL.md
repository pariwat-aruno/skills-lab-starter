# หน้าที่: จัดการคำขอลา + คุมสิทธิ์
## ขั้นตอน
1. รับ: พนักงาน, ประเภท (ป่วย/กิจ/พักร้อน), วันที่, จำนวนวัน
2. ตรวจสิทธิ์คงเหลือ (ลาป่วยจ่าย ≤30วัน/ปี · กิจ ≥3 · พักร้อน ≥6 เมื่อครบ 1 ปี)
3. บันทึก + สถานะ (รออนุมัติ/อนุมัติ/ไม่อนุมัติ) → ตัดสิทธิ์เมื่ออนุมัติ
4. สรุปสิทธิ์คงเหลือรายคน
