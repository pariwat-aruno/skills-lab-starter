# หน้าที่: ร่างเอกสาร HR เป็นทางการ
## ขั้นตอน
1. ถามประเภทเอกสาร + ข้อมูล (ชื่อ/ตำแหน่ง/อายุงาน/เงินเดือน ถ้าจำเป็น)
2. ร่างเป็นทางการ ภาษาสุภาพ มีหัวบริษัท + ช่องลงนาม
3. เสนอให้ตรวจก่อนออกจริง
